HTML MCQ Questions

  1. Which tag is used to create an ordered list in HTML?
    1. <ol>
    2. <ul>
    3. <list>
    4. <dl>

    Answer: <ol>

  2. What is the purpose of the rowspan attribute in an HTML table?
    1. It defines the number of rows a cell should span
    2. It defines the number of columns a cell should span
    3. It adds a space between rows
    4. It defines the color of table rows

    Answer: It defines the number of rows a cell should span

  3. Which HTML element is used for a definition list?
    1. <dl>
    2. <ul>
    3. <ol>
    4. <list>

    Answer: <dl>

  4. What is the purpose of the colgroup element in an HTML table?
    1. To define a group of rows
    2. To group columns for styling
    3. To add borders to table cells
    4. To define the size of table cells

    Answer: To group columns for styling

  5. How do you make a table have borders in HTML?
    1. border="1"
    2. <table border="1">
    3. <table style="border: 1px solid">
    4. All of the above

    Answer: All of the above

  6. What tag is used to define a hyperlink target in HTML?
    1. <link>
    2. target
    3. <a>
    4. <href>

    Answer: target

  7. Which HTML tag is used to display a horizontal line?
    1. <hr>
    2. <line>
    3. <br>
    4. <hrline>

    Answer: <hr>

  8. What is the default value of the list-style-type property for an unordered list in HTML?
    1. circle
    2. square
    3. disc
    4. none

    Answer: disc

  9. Which HTML tag is used to create a drop-down list?
    1. <dropdown>
    2. <select>
    3. <input>
    4. <list>

    Answer: <select>

  10. How do you add a background color to an HTML page?
    1. Using the <style> tag in the head section
    2. Using the bgcolor attribute in the <body> tag
    3. Both A and B
    4. Using CSS in an external file

    Answer: Both A and B

  11. Which attribute is used to specify the width of an image in HTML?
    1. size
    2. height
    3. width
    4. image-width

    Answer: width

  12. What tag is used to display an ordered list in HTML?
    1. <ul>
    2. <ol>
    3. <list>
    4. <dl>

    Answer: <ol>

  13. Which HTML element is used to define an inline style?
    1. <style>
    2. <link>
    3. <css>
    4. style="..." (within tags)

    Answer: style="..." (within tags)

  14. Which tag is used to define a table cell in HTML?
    1. <td>
    2. <th>
    3. <tr>
    4. <table>

    Answer: <td>

  15. How do you define a form in HTML?
    1. <form>
    2. <input>
    3. <button>
    4. <fieldset>

    Answer: <form>
